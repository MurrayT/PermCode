Description

The animation window allows you to view an animation of a selection of permutations.

Details

An animation window must be initialised with a supply of permutations, usually from another element of PermLab. For instance, the class window can generate animations, or a text window can be used to load a file and then animate the permutations it contains.

The controls are straightforward with a slider controlling the speed of the animation, various buttons that control the animation running, and a text field allowing direct entry of a frame number.

The Sort button sorts the background collection of permutations by length, and then lexicographically within length. This is sometimes useful to produce a more structured view of the collection.

There is a right-click menu available in the main display area which allows the current permutation, or all the permutations of this movie to be shown as text. The Show as PSTricks option is not currently functional.
